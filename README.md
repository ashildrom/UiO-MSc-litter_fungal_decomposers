# UiO-MSc-litter_fungal_decomposers

## Information: otu_table_mumu_curated_ashild.xlsx
### Methodological information
DNA extraction performed on 150 mg spruce litter per sample, PCR amplified and library prepared. Three libraries were paired-end sequenced using ITS2 at Fasteris, Switzerland. Raw sequences were demultiplexed using CUTADAPT, minimum overlap 26 bp, minimum read length 100 bp.
